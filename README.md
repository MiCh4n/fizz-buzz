# My fizz-buzzes
* [Python](https://github.com/MiCh4n/fizz-buzz/blob/main/main.py)
    ```
    i = 1
    while i <= 100:
        if i % 15 == 0:
            print(str(i)+" FizzBuzz!")
        elif i % 3 == 0:
            print(str(i)+" Fizz!")
        elif i % 5 == 0:
            print(str(i)+" Buzz!")
        i += 1
    ```
